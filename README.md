# Домашнее задание по теме «Коллекции TreeMap и TreeSet»

## Компаратор `PersonComparator` для объекта `Person`

### Он сортирует объект `Person` по:
1. Если у первого `Person` длина фамилии больше второго, то он помещается дальше.
2. Если у обоих `Person` длина фамилии одинакова, то проверяется возраст. Кто старше тот помещается дальше.
3. Если возраст обоих `Person` одинаков, то они остаются на месте.
